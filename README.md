# react-native-test

A React Native app built with Expo.

## Prerequisites

- Node.js installed
- npm or yarn installed
- Android Studio / Xcode if you want to run on simulator/emulator

## Install dependencies

```bash
npm install
```

## Run the project

Start the Expo development server:

```bash
npm start
```

or:

```bash
npx expo start
```

## Run on Android

```bash
npm run android
```

## Run on iOS

```bash
npm run ios
```

## Run on Web

```bash
npm run web
```

## Notes

- This project uses Expo SDK 54.
- If you do not have Expo CLI installed globally, use `npx expo` commands.
- Make sure your Android emulator or iOS simulator is running before executing the platform-specific commands.
